# TruffleScheme
Scheme (R5RS) implementation on Truffle framework.

##
Supported
- +, -, *, /
- pairs and lists
- car, cdr, cons, length, list, map*
- >, >=, <, <=, = 
- and, or
- lambda, define
- let, let*
- if, cond
- quote, quasiquote
- macros (not hygienic rn)
- eval
- current-milliseconds, display, newline
